## Installation

This project is in pre-release stage and is not yet available on crates.io. You can install it from source by cloning the repository and then running the following command from the replicator-cli folder:

```sh
cargo install --path crates/cli
```

The default binary name is `replicator`

Publishing release builds to package managers is [on our roadmap](/articles/roadmap#Technical+tasks).

## Usage

```sh
replicator --help
```
